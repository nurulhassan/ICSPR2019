Dr. Vishal Jain

Associate Professor at Bharati Vidyapeeths Institute of Computer Applications and Management, New Delh, India

  • Ph.D.
  • M. Tech.
  • MCA
  • MBA

Dr. Vishal is presently working on "Ontology Based Information Retrieval in Semantic Web", where new model has been proposed "Mining in Ontology with Multi Agent System in Semantic Web". This work focuses on the problem of choosing a representation of documents that can be suitable to induce concept-based user profiles as well as to support a content-based retrieval process. It is at this juncture that the science of Agent Mining comes to rescue. The Semantic Web addresses the first part of this challenge by trying to make the data also machine understandable in the form of Ontology, while Multi-Agent addresses the second part by semi-automatically extracting the useful knowledge hidden in these data, and making it available.
